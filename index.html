Health Tracker - устройство для оценки кардиореспираторной функции человека.

Основные функции Health Tracker включают:

Измерение пройденной дистанции

Мониторинг пульса

Определение насыщения периферической крови кислородом

Снятие электрокардиограммы (ЭКГ)

Измерение частоты дыхательных движений

Устройство Health Tracker предназначено для проведения 10-минутного теста, оценивающего функциональное состояние систем кровообращения и дыхания. При помощи этого теста определяется физическая работоспособность пациента, тяжесть заболевания (функциональный класс). Это помогает оценить состояние пациента в данный момент времени, дать прогноз заболевания, провести экспертизу трудоспособности и сделать вывод об эффективности лечения и реабилитационного потенциала.

В процессе тестирования Health Tracker фиксирует ключевые показатели:
  • Частоту пульса
  • Частоту дыхательных движений
  • Данные электрокардиографии для определения Частоты сердечных сокращений
  • Насыщение периферической крови кислородом
  • Пройденное расстояние в метрах (погрешность 0.1 м)

Эти параметры записываются на карту памяти для последующей обработки на компьютере с операционной системой Windows 10 64-bit, где они будут представлены в виде графиков, отображающих работу каждого датчика. Дополнительно предусмотрена передача данных по Bluetooth на мобильное устройство на базе Android, что позволяет врачу-оператору в реальном времени отслеживать пульс и дыхание пациента для оценки его состояния во время прохождения теста.

Трость оснащена тремя разъёмами: один type-C, предназначенный для зарядки аккумулятора, и два 3.5 мм mini jack, один из которых служит для подключения пульсоксиметра, а другой — для платы на груди. Длина трости регулируется под рост пациента. При запуске устройства происходит автоматическая проверка всех подключённых компонентов.

На встроенном мониторе отображаются подключённые части в виде иконок, таких как SD-карта, датчики ЭКГ, пульсоксиметр, маска и Bluetooth. Если какой-либо компонент не подключён, его иконка не будет отображаться, что сигнализирует о необходимости проверки соединений.

Датчик давления сделан съёмным, что упрощает следования правилам гигиены и позволяет использовать маски разных размеров.

Основная задача микроконтроллера платы, размещённой на груди пациента - сбор данных ЭКГ и давления в маске пациента с последующей передачей на основное устройство.
